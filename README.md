### ASTEROID GAME 
-----------

Simple asteroids game made in C++ and SFML library.
![ekran goruntusu](https://github.com/muhammedaltunisik/Asteroid-Game/tree/master/img)

### Dependencies
------
- SFML version 2.5.1 or higher
- C++ compiler with C++11 support.

### Commands
------
- WASD - move the ship.
- Space - shoot.
